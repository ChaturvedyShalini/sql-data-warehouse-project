# Data Warehouse and Analytics Project

Welcome to the **Data Warehouse and Analytics Project** repository.

This project demonstrates a complete **data warehousing and analytics workflow**, starting from integrating raw data into a structured data warehouse and generating meaningful business insights using SQL.

This project is designed as a **portfolio project** to showcase practical skills in **Data Engineering and Data Analytics** following industry best practices.

---

## 📊 Project Overview

This project focuses on building a **modern data warehouse using SQL Server** and performing analytics to extract valuable insights from sales data.

The workflow includes:

- **Data ingestion** from multiple sources
- **Data cleaning and transformation**
- **Data integration** into a structured warehouse
- **Analytical queries** to generate business insights

---

## 🏗 Building the Data Warehouse (Data Engineering)

### **Objective**

Develop a **data warehouse using SQL Server** to consolidate sales data and support analytical reporting and decision-making.

### **Specifications**

**Data Sources**

- Import data from two different systems:
  - **ERP System**
  - **CRM System**
- Data is provided in **CSV files**.

**Data Quality**

- Clean and handle **data quality issues**.
- Remove inconsistencies and missing values before analysis.

**Data Integration**

- Combine ERP and CRM datasets into a **single unified data model** optimized for analytical queries.

**Scope**

- The project focuses on the **latest dataset only**.
- Historical data tracking is **not included**.

**Documentation**

- Provide clear documentation of the **data model** to support both business stakeholders and analytics teams.

---

## 📈 BI: Analytics and Reporting (Data Analytics)

### **Objective**

Develop **SQL-based analytics** to generate insights that help understand business performance.

### **Key Analytics Areas**

**Customer Behavior**

- Analyze customer purchasing patterns
- Identify high-value customers

**Product Performance**

- Evaluate which products generate the most revenue
- Identify top-performing product categories

**Sales Trends**

- Analyze sales trends over time
- Support business forecasting and strategic decision-making

These insights help stakeholders monitor **key business metrics** and make **data-driven decisions**.
